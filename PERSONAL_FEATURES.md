# 개인 페이지 기능 가이드

LOPOOP의 개인 페이지 기능을 통해 공통 빚 관리와 스케줄과는 별도로 개인적인 빚과 스케줄을 관리할 수 있습니다.

## 🎯 주요 기능

### 1. 개인 빚 관리
- **공통 빚과 분리**: 길드 공통 빚과 별도로 개인 빚을 관리
- **다중 아이템 지원**: 한 번에 여러 빚 항목을 추가 가능
- **실시간 동기화**: Supabase를 통한 실시간 업데이트
- **결제 상태 추적**: 갚음/미납 상태 관리

### 2. 개인 스케줄 관리
- **다양한 스케줄 타입**: 개인, 레이드, 회의, 이벤트
- **참가자 관리**: 스케줄별 참가자 추가/제거
- **완료 상태 추적**: 스케줄 완료 여부 관리
- **시간 기반 정렬**: 시작 시간 순으로 자동 정렬

### 3. 개인 설정
- 개인화된 설정 관리 (준비 중)

## 🚀 사용 방법

### 개인 메뉴 생성

1. **메뉴 관리자**에서 "새 메뉴 추가" 클릭
2. 메뉴 이름 입력 (예: "내 개인 관리")
3. 메뉴 타입은 자동으로 "개인 페이지"로 설정됨
4. "메뉴 생성" 클릭

### 개인 빚 관리

#### 새 빚 추가
1. 개인 페이지의 "빚 관리" 탭 선택
2. "새 빚 추가" 버튼 클릭
3. 빚진 사람과 받을 사람 입력
4. 아이템 정보 입력:
   - **금액**: 골드 금액 (선택사항)
   - **아이템명**: 빚진 아이템 이름 (선택사항)
   - **설명**: 추가 설명 (선택사항)
5. 여러 아이템이 있다면 "아이템 추가" 버튼으로 추가
6. "추가" 버튼으로 저장

#### 빚 관리
- **갚음 표시**: 체크 버튼(✓) 클릭
- **수정**: 연필 아이콘 클릭
- **삭제**: 휴지통 아이콘 클릭

### 개인 스케줄 관리

#### 새 스케줄 추가
1. 개인 페이지의 "스케줄" 탭 선택
2. "새 스케줄 추가" 버튼 클릭
3. 스케줄 정보 입력:
   - **제목**: 스케줄 제목 (필수)
   - **설명**: 상세 설명 (선택사항)
   - **시작/종료 시간**: 날짜와 시간 선택 (필수)
   - **타입**: 개인, 레이드, 회의, 이벤트 중 선택
   - **참가자**: 참가자 이름 추가 (선택사항)
4. "추가" 버튼으로 저장

#### 스케줄 관리
- **완료 표시**: 체크 버튼(✓) 클릭
- **수정**: 연필 아이콘 클릭
- **삭제**: 휴지통 아이콘 클릭

## 🔧 기술적 특징

### 데이터베이스 구조
- **personal_debts**: 개인 빚 데이터
- **personal_schedules**: 개인 스케줄 데이터
- **RLS (Row Level Security)**: 사용자별 데이터 격리
- **실시간 구독**: 즉시 업데이트 반영

### 보안
- 사용자별 데이터 격리
- 본인 데이터만 접근 가능
- 안전한 데이터베이스 정책

### 성능
- 인덱스 최적화
- 효율적인 쿼리
- 실시간 동기화

## 📋 데이터베이스 마이그레이션

개인 페이지 기능을 사용하려면 데이터베이스 마이그레이션이 필요합니다:

```bash
# 개인 테이블 마이그레이션 실행
pnpm migrate:personal

# 또는 수동으로 실행
ts-node scripts/apply-personal-migration.ts
```
